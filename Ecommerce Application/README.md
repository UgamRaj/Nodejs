# Api template

## Base URl

`https://nodejs-1-puyy.onrender.com/`

## Endpoints

#### User

`POST/api/user/register`

`POST/api/user/login`

`GET/api/user/allusers`

`DELETE/api/user/:id`

`PUT/api/user/edituser`

`GET/api/user/logout`

`PATCH/api/user/resetpassword`

### Product

`POST/api/product`

`GET/api/product/:id`

`PUT/api/product/:id`

`DELETE/api/product/:id`
